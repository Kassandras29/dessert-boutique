# Интеграция дополнительных сторонних JS-библиотек и фреймворков

## Выбранные библиотеки
- **marked** — парсинг Markdown в HTML (используем для отображения текста динамических отзывов)
- **DOMPurify** — очистка HTML после парсинга Markdown (защита от XSS, чтобы пользователь не мог вставить опасный HTML/JS)

## Вариант использования
На странице **reviews.html** пользователь вводит текст отзыва в поле textarea
Теперь этот текст можно писать с Markdown-разметкой:
- `**жирный**`, `*курсив*`
- списки `- пункт`

При выводе отзывов:
1) `marked` превращает Markdown в HTML
2) `DOMPurify` очищает HTML от опасных элементов
3) В DOM вставляется уже безопасный HTML
