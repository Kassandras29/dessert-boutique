# Dessert Boutique

Готовый проект, сделанный в рамках выполнения лабораторных работ по frontend-разработке.

Сайт кондитерской **Dessert Boutique** — это веб-приложение с пользовательским интерфейсом для демонстрации ассортимента и взаимодействия с пользователем.

---

## О проекте

Проект представляет собой интерфейс сайта кондитерской, разработанный с акцентом на удобство использования и визуальную привлекательность.

---

## Используемые технологии

- JavaScript
- HTML
- CSS

---

## Функциональность

- отображение ассортимента
- отображение и добавление отзывов
- пользовательский интерфейс для взаимодействия
- адаптивная верстка
